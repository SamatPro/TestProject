Перед запуском проекта:
1)распаковать zip архив
2)указать в db.properties свой логин и пароль, url БД
3)выполнить скрипт department_backup.sql
4)убедиться, что соединение с БД успешно

Запуск проекта:
Открыть командную строку в директории TestProject
Выполнить > startp.bat %команда %имя_файла

Примечание!
Команда sync - синхронизирует данные из XML в БД
        load - загружает данные из БД в файл
Имя файла должно быть указано без расширения!
При выполнении sync обязательно убедитесь,
что указали существующее имя и
корректный файл.
